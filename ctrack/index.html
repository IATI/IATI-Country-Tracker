Country Tracker aims to be a simple user interface to search and browse all IATI data relating to a specific country that can be easily hosted by anyone, anywhere.

The final Country Tracker product will be available as a set of HTML, Javascript, and CSS files.

Since HTML/CSS/Javascript are the technologies of the web, and are widely used and understood, we believe this is a good option for easy adoption by others, for easy deployment, and for easy re-configuration/customisation.

SAVi is a simple activity viewer for IATI XML that you can view in real time, online, in your favourite browser.

SAVi uses CSS and javascript to display otherwise complex-looking XML into a format that is easier to read and digest.

dstore subsidizes the IATI Datastore with an optimized nodejs + SQLite version for use in real time Country Tracker queries.
Q allows you to query information you need via simple but complex filters.
Here are some Query examples:
http://dev.ctrack.iatistandard.org/q.xml?from=activities&limit=10&orderby=day_start&day_start_gt=1950-01-01
Returns XML which is automatically formatted into HTML, source remains pure XML
This query displays the activities from=activities limiting the amount to 10 limit=10 sorted by starting day orderby=day_start and from 1950 onwards day_start_gt=1950-01-01.
http://dev.ctrack.iatistandard.org/q?groupby=reporting_org&select=stats
Returns JSON (JSON is default) data about activites (activites are default) where 100 rows from the database is the default limit
This query displays data statistics select=stats about activites for each reporting organisation groupby=reporting_org.
http://dev.ctrack.iatistandard.org/q.csv?from=transactions&orderby=day&limit=1000
Returns CSV file (named q.csv by default)
This query allows you to download the CSV q.csv of 1000 limit=1000 transaction data from=transactions sorted by date orderby=day.
