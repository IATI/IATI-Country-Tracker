The updated EXPLORE feature helps find activities published to the IATI Standard using filters and keywords.

Updated features following development in the first half of 2017 include:

  • Choose multiple options on all filters to allow much more detailed and refined searches
  • IATI Activity status included as an option within filters
  • Search for an individual activity ID using the freetext box
  • New toggle button to sort filter dropdowns by alphabetical order or by relevant IATI/CRS code (e.g. by CRS sector numbers)
  • New EXPLORE section within the FAQ page specifically relating to the search function

Development is ongoing - as always feedback (email or GitHub) is very welcome.
